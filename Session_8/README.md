
## Session 8 - RECEPTIVE FIELDS & NETWORK ARCHITECTURES
***

> ##### Assignment Objective
> * Extract Resnet18 model from pytorch-cifar repo
> * Train Resnet18 on CIFAR10 dataset
> * Achieve 85% target accuracy
> * Use any number of epochs

***
> ##### Modular Implementation
> * model_architecture.py - GBN implementaion, architecture creation and view
> * train_test.py - train and test functions
> * resnet.py - Resnet18 model from pytorch-cifar repo

***
> ##### Model Summary
> * The model reaches a test accuracy of 90% in CIFAR-10 dataset in 22 epochs. The model has 11,173,962 parameters.


> ##### Hyperparameter
> - Loss Function: Cross Entropy Loss
> - Optimizer: SGD
> - Dropout: 0.10
> - Batch Size: 128
> - Number of epochs: 30
> - Learning Rate: 0.016
> - Ghost Batch Norm Splits: 2
> - LR Scheduler: Step LR with step size of 2 and gamma of 0.92
